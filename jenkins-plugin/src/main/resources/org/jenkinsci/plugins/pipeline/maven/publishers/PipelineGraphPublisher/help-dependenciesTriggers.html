Skip Downstream Triggers:Don't trigger this pipeline if a maven dependency declared in the withMaven block has changed, even if the pipeline has registered a trigger on SNAPSHOT dependencies changes.
This flag is convenient to prevent storms of pipeline builds when modifying a widely mutualized maven artifact.
