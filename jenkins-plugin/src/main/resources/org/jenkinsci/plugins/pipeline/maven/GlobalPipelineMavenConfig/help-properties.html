Add a property (name/value pair) that will be used to configure the DataSource/Driver.

In the case of a DataSource, the property names will be translated to Java setters following the Java Bean naming convention. For example, the property cachePrepStmts will translate into setCachePrepStmts() with the value passed as a parameter.

In the case of a Driver, the property will be added to a Properties instance that will be passed to the driver during java.sql.Driver#connect(String, Properties) calls.
