Sélectionner un élément Maven globale settings parmi les fournisseurs disponibles. L'élément settings dans le fichier settings.xml contient les éléments utilisés pour définir des valeurs qui configurent l'exécution Maven de plusieurs façons, comme le pom.xml, mais qui ne devraient pas être embarquées dans un projet spécifique, ou distribuées publiquement. Ces valeurs incluent l'emplacement du mirroir local, un serveur distant alternatif et des identifiants de connexion.
