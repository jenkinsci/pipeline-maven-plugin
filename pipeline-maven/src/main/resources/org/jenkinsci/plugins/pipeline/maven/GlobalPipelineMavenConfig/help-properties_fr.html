Ajouter une propriété (paire nom/valeur) qui sera utilisée pour configurer le DataSource Hikari CP / le Driver JDBC.

Pour le cas du DataSource, les noms de propriété doivent être préfixés par dataSource., ces propriétés seront traduites en setters Java selon la convention de nommage Java Bean. Par exemple, la propriété dataSource.cachePrepStmts sera traduite en setCachePrepStmts() avec la value passée comme paramètre. Voir les détails sur Hikari CP.

Dans le cas du Driver, la propriété sera ajoutée à une instance Properties qui sera passée au pilote durant les appels java.sql.Driver#connect(String, Properties).

Pour une connexion MySQL/MariaDB, le plugin Pipeline Maven utilisera par défaut les paramètres recommandés dans la configuration Hikari CP / MySQL. Pour surcharger ces paramètres, il suffit de les ajouter dans le champ de configuration, avec des valeurs différentes.

Pour les bases autres que MySQL/MariaDB, le plugin utilise les paramètres par défaut du data source Hikari CP.
