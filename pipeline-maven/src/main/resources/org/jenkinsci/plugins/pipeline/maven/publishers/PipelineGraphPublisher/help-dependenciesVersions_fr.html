Type des versions, "release" ou "snapshot", des dépendances Maven surveillées pour déclencher cette construction. Par exemple, il n'est peut être pas souhaitable de déclencher une construction si une dépendance "release" a été modifiée.
Exclure les dépendances "release" est une optimisation pour réduire la taille de la base de données contenant le graphe des projets et livrables Maven.
