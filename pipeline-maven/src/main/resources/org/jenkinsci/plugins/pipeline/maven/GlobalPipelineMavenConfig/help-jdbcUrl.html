Supported databases are H2, MySQL(including MySQL 5.7+, MariaDB 5.5+, Amazon Aurora DB for MySQL 5.6+...) and PostgreSQL up to 16. For production workloads, use MySQL / MariaDB or PostgreSQL.

To use MySQL, install the MySQL Database Plugin to load the MySQL driver.

To use PostgreSQL, install the PostgreSQL API Plugin to load the PostgreSQL driver.

If empty, then a non production grade H2 embedded database will be used.
