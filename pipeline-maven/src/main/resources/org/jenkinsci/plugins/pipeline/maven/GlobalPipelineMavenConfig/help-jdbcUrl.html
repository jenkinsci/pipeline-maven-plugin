Supported databases are H2, MySQL(including MySQL up to 8.1, MariaDB up to 11.1, Amazon Aurora DB for MySQL 5.6+...) and PostgreSQL up to 16. For production workloads, use MySQL / MariaDB or PostgreSQL.

To use MySQL, install the MySQL Database Plugin to load the MySQL driver.

To use PostgreSQL, install the PostgreSQL API Plugin to load the PostgreSQL driver.
