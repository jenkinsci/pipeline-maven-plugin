Skip Downstream Triggers: Don't trigger dependent pipelines if a maven artifact declared in the withMaven block has changed, even if those pipelines have registered a trigger on SNAPSHOT dependencies changes.
This flag is convenient to prevent storms of pipeline builds when modifying a widely mutualized maven artifact.

Ignore Upstream Triggers: Don't trigger this pipeline if a maven dependency has changed by its own pipeline, even if this pipeline has registered a trigger on SNAPSHOT dependencies changes.
