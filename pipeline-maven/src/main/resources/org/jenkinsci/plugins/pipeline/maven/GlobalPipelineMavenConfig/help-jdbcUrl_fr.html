Les bases de données supportées incluent H2, MySQL (MySQL jusqu'à la version 8.1, MariaDB jusqu'à la version 11.1, Amazon Aurora DB for MySQL 5.6+...) et PostgreSQL jusqu'à la version 16. Pour un environnement de production, préférer MySQL / MariaDB ou PostgreSQL.

Pour utiliser MySQL, installer le plugin MySQL Database pour charger le pilote MySQL.

Pour utiliser PostgreSQL, installer le plugin PostgreSQL API pour charger le pilote PostgreSQL.
